Vidimost-dl
===========

Application to retrieve data.


Requirements
============

* git bash
* python 3
* virtualenv (pip)
* virtualenvwrapper / virtualenvwrapper-win (pip)
* doit (pip)


Start
=====

* mkvirtualenv vidimost-dl / workon vidimost-dl
* doit install
* doit docker_build


Launch
======

* doit list
* doit launch
* doit uvicorn
* doit docker_run
